🏍️ Sistema de Gestión de Inventario – Accesorios para Motos
Aplicación web para la gestión de inventario de una tienda de accesorios para motos.
Permite llevar el control de productos, stock y movimientos de inventario de forma sencilla y eficiente.

El proyecto está desarrollado con PHP para el backend, JavaScript para el frontend y MySQL como sistema de base de datos, todo ejecutándose en un entorno Dockerizado.

🚀 Características principales
📦 Registro y gestión de productos
📊 Control de stock en tiempo real
➕➖ Entradas y salidas de inventario
🔍 Búsqueda y filtrado de productos
🧾 Organización por categorías
🌐 Aplicación web accesible desde el navegador

🗄️ Base de Datos
La base de datos utiliza MySQL y se encuentra alojada en un contenedor Docker, lo que facilita la portabilidad y configuración del entorno.

Inventario
